Project Order - Front-End
=============

# Installation

	$ cd Orders
	$ npm install
    $ ng serve

# Description	

Project to made, filter, and show orders 

Project contains:
* Material Design
* Grid Usign NG2
* Angular CLI
* [BackEnd](https://github.com/rdiegoss/orders_back)
